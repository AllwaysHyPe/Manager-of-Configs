HCP Terraform + Microsoft Azure workshop
Abstract

The fundamental building block of modern infrastructure deployments is a strong basis of codification.

In this workshop, Steve and Kerim will take you from Zero to Managed with HashiCorp Terraform and Microsoft Azure.

In this hands-on one hundred and twenty minute session you can expect to learn:

How to manage infrastructure with Terraform
How to safely inject credentials into the process
How to think about scalability and governance

You'll leave with new insights and a solid understanding of the complete provisioning process, no matter if you're working with existing (brownfield) infrastructure or are rolling out a new platform, greenfield-style, for the first time.


Description
The fundamental building block of modern infrastructure deployments is a strong basis of codification.

In this hands-on workshop, geared towards operations engineers (DevOps, SRE, Platform), Steve Murawski and Kerim Satirli will take you from Zero to Managed with HashiCorp Terraform and Microsoft Azure.

From understanding Terraform's core workflow to deploying and configuring local and remote infrastructure, this workshop is jam-packed with all the skills and techniques needed to effectively manage infrastructure as code, with Terraform.

In this hands-on ninety minute session you can expect to learn:

How to manage infrastructure with Terraform
How to safely inject credentials into the process
How to think about scalability and governance

Attendees will get access to a full environment to build along.


Notes

For the best attendee experience, here are the prerequisites: 

Attendees will need a GitHub account. 
Attendees will need to bring a laptop with the latest version of the Terraform CLI pre-installed
Attendees can alternatively use GitHub Codespaces to spin up a compatible environment
A basic understanding of Linux and interacting with CLI applications
